RocketConf 2015

A three-day single track conference for modern web developers

11, 12 and 13 November, 2015 — Amsterdam, The Netherlands

Awesome! You’re here!

This November we will bring you RocketConf, a three-day developer conference for the modern web developer and dev ops who loves writing beautiful code and creating magnificent things that matter.

RocketConf does not focus on any particual technology or tool. We believe that you can learn more from people with a different perspective. Technical topics can range from Ruby on Rails to Ember.js to Elixir to Docker and everything in between.

To learn something well, you should get hands-on experience. On the 11th November we'll host workshops where you can get together with others and get a real learning experience by doing something fun. The 12th and 13th November are reserved for the conference talks.

Talking tech in beautiful Amsterdam
